# Setup

## Local bring-up checklist

1. Serve the `web/` directory with a local static server.
2. Open the scene in a desktop browser for quick checks.
3. Deploy the site to an HTTPS host for headset testing.
4. Open the deployed URL in Meta Quest Browser.
5. Enter immersive VR and confirm tracking and interaction basics.

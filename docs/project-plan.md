# Project Plan

## Phase 1

Stand up a working Quest 3 web prototype repo with durable context and a minimal deployable scene.

## Phase 2

Choose a tiny app concept, such as:

- interactable room-scale toy
- spatial sketch pad
- one-room exploration prototype

## Definition of done for the first playable

- Runs in Meta Quest Browser on Quest 3
- Can be opened from a shared link
- Has basic fixed-position interaction
- Includes at least one meaningful interaction
- Build and run steps are documented
- Repo context is recoverable from markdown alone
